# -New-s-Scraper-2


``This web app allowes users view and leave comments on the latest New York Times news. By using  Mongoose and Cheerio to scrape news from latest New York Times World articles.``



``Technologies Used:
	express
	express-handlebars
	mongoose
	body-parser
	cheerio
	request
``
